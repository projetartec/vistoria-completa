# **App Name**: FireCheck Brazil

## Core Features:

- Offline Functionality: Enable offline functionality for creating, saving, and loading inspection checklists on mobile devices without requiring an active internet connection.
- Customizable Header: Include header displaying company name (BRAZIL EXTINTORES) and document title (VISTORIA TÃ‰CNICA), along with the company logo on the left.
- Automatic Inspection Numbering: Automatically generate inspection numbers based on a customer code and sequential numbering system, linking each inspection for easy retrieval and reference.
- Client Data Capture: Capture client data including location, client code, and floor number for each inspection, ensuring detailed record-keeping.
- Inspection Checklist: Provide checklist items for extinguisher, fire door, alarm, lighting, and hydrant inspections, with sub-items and fields for OK, N/C, and N/A checks, plus observation notes, also provide specific items to test the water pumps related to the sprinklers (SPK) and the hydrants, measure water pressure in each system, also create options for showing and hiding each inspection subitem or note as the user sees fit. Provide options for adding extinguishers and firehose as needed and link each added firehose and fire extinguisher to the location they are used. Display data of each linked firehose and fire extinguisher to each saved file of the technical inspection.
- Collapsible Sections: Implement expandable and collapsible sections for each checklist item, allowing users to focus on specific areas of the inspection while maintaining an organized layout.
- PDF Report Generation: Enable users to download inspection reports as PDF files, incorporating header data, floor number, the company's logo, firehose and extinguisher registry, and all inspection details for comprehensive documentation.

## Style Guidelines:

- Primary color: Soft, desaturated blue (#64B5F6) to convey a sense of trust and safety, critical in fire safety applications.
- Background color: Very light gray (#F5F5F5) for a clean, unobtrusive backdrop.
- Accent color: Pale cyan (#A7FFEB) to highlight interactive elements and important actions.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet accessible and readable text.
- Use clear, universally recognized icons for each inspection item.
- Ensure a clear and intuitive layout with consistent spacing and alignment to enhance usability.
- Subtle animations when expanding or collapsing checklist items to provide a smooth user experience.
